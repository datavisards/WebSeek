# WebSeek

A web extension that streamlines web data analysis by integrating a data-instance canvas with AI agents.


## Development

```
npm install
npm run dev
```

Open `chrome://extensions` in Chrome, turn on the developer mode, click on `Load unpacked`, and load `.output/chrome-mv3-dev` to Chrome.

## Deployment

```
npm run build
```

Open `chrome://extensions` in Chrome, turn on the developer mode, click on `Load unpacked`, and load `.output/chrome-mv3` to Chrome.
